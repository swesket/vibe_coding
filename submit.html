사이트 등록 요청

운영 중인 패션 쇼핑 사이트를 DIGGING에 등록하고 싶으시다면 아래 양식을 작성해주세요. 검토 후 연락드리겠습니다.

DIGGING은 신뢰할 수 있는 패션 쇼핑 사이트만을 엄선하여 소개합니다. 등록을 원하시는 사이트는 DIGGING의 4단계 검증 프로세스(에디터 방문 평가 → 실제 주문 테스트 → 반품·교환 확인 → 커뮤니티 평판 분석)를 거치게 됩니다. 검증을 통과한 사이트는 6개 카테고리(국내 편집숍, 럭셔리, 스트릿, 빈티지, 스니커즈, 글로벌) 중 적합한 곳에 배치되며, 4개 국어(한국어, 영어, 중국어, 일본어)로 소개됩니다. 등록 심사는 약 2주 정도 소요되며, 결과는 입력하신 이메일로 안내드립니다.

📝

1. 양식 제출

아래 양식에 사이트 정보를 입력하고 제출해주세요. 사이트 이름, URL, 카테고리, 소개글은 필수 항목입니다.

🔍

2. 에디터 검토

DIGGING 에디터가 제출된 사이트를 직접 방문하여 상품 퀄리티, UI/UX, 배송 정책, 고객 서비스를 종합적으로 평가합니다.

✅

3. 등록 완료

검토 기준을 충족하면 DIGGING 디렉토리에 등록되며, 결과는 이메일로 안내드립니다. 통상 7~14일이 소요됩니다.
